# XTB

## Overview 

```
 XTB
   [ACC <real acc default 1.0>] 
   [UHF]
   [METHOD gfn1 || gfn2 default gfn2]
   [ETEMP]
   [GUESS]
   [VERBOSITY]
   [PRINT]  
   [NOPRINT]
   [NSPIN <integer nspin default 1>]

 END
```
## ACC: Accuracy

For example, if the user want to increase the accuracy:
```
 acc 1d-4
```
